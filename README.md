# Beauty Intelligent Wellness Platform

A premium clinic operations platform for Beauty Intelligent Wellness in Bashundhara.

## Stack
- Backend: FastAPI
- Frontend: Next.js
- Database: Supabase
- Backend hosting: Render
- Frontend hosting: Vercel

## Areas
- CRM
- ERP
- Sales
- Revenue
- Cost tracking
- Branch dashboards
- Employee service assignment
- Bonus and payroll automation

## Structure
- `backend/` - FastAPI API
- `frontend/` - Next.js app
- `docs/` - implementation plan and architecture notes
